# Base Nodejs Project

### Requirement

- Nodejs : ^14.18.1
- NPM : ^6.14.15

### Installation

- Clone repository
```sh
git clone https://github.com/oktapiancaw/base-nodejs-project
```

- Install library & module
```sh
npm install
```

- Run the project
```sh
# Production
npm run build

# Development
npm run start
```
